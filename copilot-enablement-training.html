Part 1：What is Microsoft Copilot?

⏱ 10分钟 · Copilot 是什么？

1.1 从传统工具到 AI 助手

过去

Word = 写文档
Excel = 算数据
Teams = 沟通

现在

Copilot 成为工作伙伴。
它理解你的工作上下文，帮你更快地完成从信息整理到内容创作的全过程。

1.2 Copilot 不是 ChatGPT

ChatGPTMicrosoft Copilot
知识公共知识企业知识 + 公共知识
数据开放数据Microsoft 365 数据
权限个人环境企业权限体系
目标回答问题辅助工作

1.3 Copilot 三个核心能力

Understand理解内容、上下文、工作目标
→
Create生成文档、分析、汇报
→
Act执行流程、任务、Agent 行动

Part 2：Microsoft 365 Copilot Work Scenarios

⏱ 25分钟 · 从一天工作流程理解 Copilot

不按 App 列表讲——按员工工作过程讲。

📧 Outlook

场景1：信息管理

业务问题："我每天收到大量邮件，如何快速理解重点？"

Copilot：总结邮件线程 · 提取关键决定 · 生成回复

价值：减少信息处理时间，不再被邮件淹没。

💬 Teams

场景2：团队协作

业务问题："会议结束后，如何确保事情推进？"

Copilot：Meeting Summary · Action Items · Decision Tracking · Chat Summary

关键观点：Teams 不是聊天工具——它是未来数字工作入口。

📝 Word

场景3：内容创作

场景：创建 Proposal · Report · SOP · 培训材料

能力：Draft（初稿）→ Rewrite（改写）→ Summary（摘要）

价值：从空白页到成文，Copilot 帮你迈出第一步。

📊 PowerPoint

场景3b：汇报生成

场景：从 Word 报告直接生成 Management Presentation

能力：自动抽取要点 → 匹配模板 → 生成幻灯片

价值：告别"复制粘贴做PPT"的重复劳动。

📈 Excel

场景4：业务分析

场景：销售数据分析

Copilot 帮助：发现趋势 · 解释数据 · 创建分析图表

价值：不需要精通公式，用自然语言提问即可获得洞察。

📁 OneDrive / SharePoint

场景5：知识与研究

场景：整理个人和团队知识

能力：文件问答 · 跨文档检索 · 知识关联

引出：下一阶段——企业需要 Knowledge Foundation。

Part 3：Enterprise Knowledge Foundation

⏱ 10分钟 · 为什么企业 AI 需要知识基础？

3.1 普通 AI vs 企业 AI

普通 AI

知道世界。

通用知识、公开数据、无企业上下文。

企业 AI

需要知道：
我们是谁 · 我们做过什么 · 谁拥有经验 · 哪些方案成功

3.2 Knowledge Journey

→

→
AI Intelligence可用智能

3.3 BS.A West Knowledge Platform Architecture

六层企业知识平台架构

Part 4：Copilot Agent & Copilot Studio

⏱ 10分钟 · 从 AI 助手到数字员工

4.1 什么是 Agent？

Agent 不是聊天机器人——它是具有知识、目标和行动能力的 AI 助手。它可以理解特定业务场景、访问指定数据源、执行预定义任务。

4.2 Agent 三种方式

方式一

Pre-built Agent

微软提供，开箱即用

如 Research Agent、Visual Creator Agent

👤 所有用户

方式二

Agent Builder

业务用户快速创建

FAQ Agent · Knowledge Agent · SharePoint Agent

👤 业务人员

方式三

Copilot Studio

企业级 Agent 部署

多系统连接 · 自动流程 · 业务执行

👤 IT 和专业团队

4.3 Agent 的组成要素

Instructions指令
+
Knowledge知识来源
+
Connectors数据连接
+
Guardrails行为边界
+
Governance治理

4.4 Agent 应用案例

Sales Agent

知识：Customer · Project · Service

能力：客户准备 · Proposal 支持

Project Agent

能力：项目总结 · 风险分析 · Lessons Learned

Marketing Agent

能力：内容生成 · 市场分析
