三种执行方式

方式一：Journey Kit

自建邮件序列

下载可定制的邮件模板（.OFT 格式）本地化通过公司邮件系统发送。适合有 IT 支持的组织。当前可用。

方式二：Microsoft 直发

Microsoft 代发邮件

管理员在 Admin Center 开启Microsoft 直接向终端用户发送 6 封周报邮件。零配置。完整版 2025.3 可用，Chat 版 2025 夏可用。

方式三：线上培训活动

Virtual Training Event

参加或邀请用户参加 Microsoft 官方的线上培训——获取灵感和技巧。适合作为 Journey 的补充。

莱茵建议：优先方式一（Journey Kit）——完全掌控邮件内容、发送节奏和本地化。方式二作为补充（管理员开启后 Microsoft 自动发，零维护成本）。方式三适合 Journey 结束后持续学习。

Week 1 · MondayCopilot Chat · Get the Latest Information

教学目标

用户第一次打开 Copilot Chat并完成对话。不追求完美 Prompt——追求"我做到了"的成就感。这是整个 Journey 最重要的 Day 1。

操作步骤

  1. 打开 Copilot Chat（m365.cloud.microsoft/chat）
  2. 输入 Prompt
  3. 查看 Copilot 返回——它列出你的邮件、聊天、文件活动
  4. 关键体验：用户第一次看到 AI 能访问自己的真实工作数据

官方 Prompt（Overview PDF）

Activity
Get the latest information with Copilot Chat
Prompt
"What happened this week? Summarize what I worked on, organized by emails, chats, and files."
莱茵本地化
"总结我这周做了什么，按邮件、聊天和文件分类整理。"

邮件内容

"Welcome to the Great Copilot Journey! Through this 6-week interactive email series, you will learn all the ways Microsoft 365 Copilot can empower you. Ready? Open Copilot Chat and try your first prompt."

教师行动：发送邮件当天在群里问"它总结出什么你忘了的事？"——社交证明驱动打开率。如果有领导层的欢迎信（Step 2 Kick-Off），打开率可提升 2-3 倍。

Week 1 · TuesdayTeams · Summarize What Happened

教学目标

用户在真实 Teams 会议中使用 Copilot——不是演示，是真的在自己参加的会议中用。

Week 1 · WednesdayWord · Summarize a Document

教学目标

用户体验 Copilot 在 Word 中内嵌工作——不需要复制粘贴到 Chat，直接在文档内总结。这是用户第一次看到"应用内 AI"的体验。

操作步骤

  1. 打开任意较长的 Word 文档
  2. 点击 Home 标签下的 Copilot 图标
  3. 输入 Prompt 或从建议中选择
  4. 关键体验：不需要复制粘贴——Copilot 直接读取当前文档

官方 Activity（Overview PDF）

Activity
Summarize a document — Word
Prompt
"Summarize this document. Give me 3 key takeaways."
莱茵场景
"总结这份审核报告的核心发现和不合格项。"
"这份合同的风险条款有哪些？如果延迟交付，最大赔偿是多少？"

教师提示：建议用户找一份真实的、让他们头疼的长文档来测试——真实场景比演示素材有说服力 10 倍。强调：Copilot 能直接理解文档内容，不需要你先"教会"它。

Week 1 · ThursdayOutlook · Reply to an Email

教学目标

用户在Outlook 中用 Copilot 起草邮件回复——日常最高频场景。每天至少省 20 分钟。

操作步骤

  1. 在 Outlook 中打开一封邮件
  2. 点击 Reply 点击 Copilot 图标
  3. 告诉 Copilot：立场 + 结构 + 语气

官方 Activity（Overview PDF）

Activity
Reply to an email — Outlook
Prompt
"Reply: thank for the proposal, agree with points 1-2, need more discussion on point 3. Tone: polite but firm."
莱茵本地化
"回复：感谢方案+同意前两点+第三点需讨论（提替代方案）+语气礼貌但坚定。回复开头附原邮件三点摘要。"

教师提示：Outlook Copilot 的关键公式：给立场 + 给结构 + 给语气。只说"帮我回复"结果随机。好指令 = "回复：感谢+同意前两点+对第三点提替代方案，语气礼貌但坚定"。建议用户在回复一封真实邮件时使用。

Day 29 · Dream Big 终极项目 — 完整操作指南

Source: The Great Microsoft 365 Copilot Journey Day 29 project instructions.pdf · 10 pages
全程在一个 Copilot Chat 对话中完成 · 约 20 分钟 · 不可拆分到多个对话（会丢失上下文）

01 The Concept · 点燃创意

自己先想好三要素：一个常见的场所 + 一种风格主题 + 一个出人意料的地点。然后让 Copilot 帮你扩展创意。

Copilot 技能：Kickstarting creativity · Setting context

Prompt："I'm working on a project that will transform a common place into a new and exciting destination. Give me a creative list of 10 ideas that each include a common place, a creative aesthetic theme, and a creative location like outer space, a fairytale land, or a rainforest."

02 The Context · 定义 Copilot 的角色

给 Copilot 一个"职位描述"——它将成为你的专属体验设计师。

Copilot 技能：Providing context for Copilot

Prompt："You are an extraordinary experiential designer who can craft immersive destinations. You think creatively, pay attention to every detail, and always deliver unique results."

03 The Exterior · 外观设计

让 Copilot 了解你的目的地创意→生成外观描述→生成外观图片→下载你最喜欢的。

Copilot 技能：Defining tone and purpose · Inclusions / exclusions · Creating an image

Prompt 1："The project will be to create a [your place] with the theme of [your theme] in [your location]. [Add details]. Create a paragraph with a travel marketing tone and embellish the details. Do not give the idea a name but include what makes it unique."

Prompt 2："Create an image that visualizes the exterior of this destination from a distance."

04 The Entrance · 入口设计（第一印象）

入口是访客的第一印象。脑暴5个创意→选一个展开→旅行杂志风格改写→生成图片→下载。

Copilot 技能：Contextual directives · Follow-up · Summarizing · Tone setting · Creating image

Prompt 1："Let's think about the entrance for this destination. It should embrace the theme and be a memorable experience. Suggest five ideas. This part is called The Entrance."

Prompt 2："I like the idea of [choose one]. [Add details]. Can you elaborate?"

Prompt 3："Rewrite that description in a tone you'd find in a travel magazine. Keep it to a single paragraph."

Prompt 4："Create an image that visualizes the description of the entrance you just wrote."

05 The Feature · 核心空间（WOW因子）

设计主空间——氛围、座位、窗户、灯光。表格格式列出5个空间创意→选一个展开→生成图片。

Copilot 技能：Formatting responses · Tone and context · Kickstarting creativity · Creating image

Prompt 1："We are now going to work on The Feature. Suggest 5 unique interior features or spaces. List them with: ## Name ## Description."

Prompt 2："Let's work on visualizing the idea called [choose one]. Rewrite in a travel publication tone. Also suggest five alternative names."

Prompt 3："I like the [name] you suggested. Create an image to visualize that idea."

06 The Food · 菜单设计

为你的空间设计招牌菜单——表格格式列出5道菜（名称+描述+食材+摆盘）→选一道生成图片。

Copilot 技能：Reference context · Formatting as table · Defining result · Brainstorming · Creating image

Prompt 1："For this destination, let's think about food options. This part is called The Food. Create a menu formatted as a table — name, description (ingredients + how served), theme. Provide 5 ideas."

Prompt 2："Create an image of the [choose one dish] menu option you suggested."

07 The Drink · 招牌饮品

食物配什么喝？设计5款主题饮品（强调容器/杯具的独特性）→选一款生成图片。

Copilot 技能：Reference context · Define outcome · Brainstorm · Creating image

Prompt 1："For this destination, let's think about a signature drink. This part is called Signature Drink. It should embrace the theme and surprise people. What it's served in matters — the glass should be unique. Give me five ideas."

Prompt 2："Create an image to visualize the idea you suggested called [name of your chosen drink]."

08 The Music · 氛围音乐

音乐定调——让 Copilot 生成一张10首歌的表格（艺人+歌名），贴合空间主题。

Copilot 技能：Ideation · Setting context · Formatting as table

Prompt："We need to think about music. This part is called The Music. Create a table of 10 songs with the Artist and Song Name that would fit with the theme of this destination."

09 The Description · 品牌命名与描述

给你的目的地取5个名字+tagline→选一个→生成旅行杂志风格的三句话描述。

Copilot 技能：Ideation · Summarizing · Setting tone · Defining result

Prompt 1："Given everything we have worked on, it's time for branding. Suggest 5 options for a catchy name for this destination along with a short one-sentence tagline."

Prompt 2："I like the [choose name] name. Create a three-sentence description in the tone of a travel publication that captures the essence of this destination."

10 Finish & Share · 整合发布

让 Copilot 回顾整个对话，输出所有部分的最终版本→点击"Edit in Pages"→在每节插入对应图片→分享给团队。

Copilot 技能：Reference previous work · Defining output format · Creating a Page

Prompt："Let's review everything you wrote. Give me the latest edited copy: Introduction, entrance, feature, food, signature drink, music, branding. Format: ## Title ## Description"

在 Pages 里逐节插入图片：光标定位→输入"/image"→选择已保存的图片。

教师提示：这个项目覆盖 Copilot 全部核心技能——创意激发、上下文管理、格式化输出、多轮细化、图片生成、文档整合。完成后的作品是学员最好的"毕业作品"，可内部展示/评选。不限格式——学员也可用 PowerPoint 或 Word 做最终呈现。

Pre/Post Journey Survey & DAU Monitoring 操作指南

Source: Overview PDF Page 5 "The Process (pilot phase)" — Setup & Kick-Off & Follow Up steps

Pre-Journey Survey（启动前）

测量用户对 Copilot 的初始感知，建立对照基线。

  • 你对 Copilot 的了解程度？（1-5）
  • 你目前使用 Copilot 的频率？（每天/每周/每月/从未）
  • 你最期待 Copilot 帮你解决什么问题？（开放）
  • 你对 AI 工具的态度？（积极/中立/怀疑/抵触）

工具：Microsoft Forms / SurveyMonkey · 在 Kick-Off 邮件中附链接 · 设置 3 天内截止

Post-Journey Survey（结束后）

在 Day 30 发送，测量变化 + 收集团队反馈。

  • 你对 Copilot 的了解程度？（对比 Pre）
  • 你现在使用 Copilot 的频率？（对比 Pre）
  • Journey 中最有用的 3 个 Prompt？（多选）
  • 你有多大可能向同事推荐？（NPS 0-10）
  • 你对下一期 Journey 的建议？（开放）

配合 Badge 发放一起推送 · 回复率通常比 Pre 高（用户已有参与感）

DAU Baseline Monitoring

在 Setup 阶段从 Microsoft Copilot Usage Report 拉取基线数据。

  • 抓取时间点：Journey 开始前 1 周
  • 核心指标：Daily Active Users (DAU) / Weekly Active Users (WAU)
  • 入口：Microsoft 365 Admin Center → Reports → Copilot Usage
  • 对比节点：W2 / W4 / W6 各拉一次 → 形成趋势图

Champion Community 启动法

Follow Up 阶段：从 Journey 参与者中识别高活跃用户 → 建群。

  • 识别标准：完成 ≥80% 天任务 + Post-Survey NPS ≥8
  • 启动动作：Leadership 发邀请信 → 建 Teams 频道 → 首月每周 1 次 Office Hour
  • 长期运营：Prompt of the Week / 用户案例分享 / 内部 Badge 体系
  • 目标：3 个月内 Champion 人数达到初始参与人数的 15-20%

Source: Overview PDF Page 5 Closing & Follow Up — "Identify engaged users to start your own Copilot Champion Community"

数据闭环：Pre-Survey(基线) → DAU(W2/W4/W6趋势) → Post-Survey(变化量) → 3个月后追踪DAU(持续效应)。这套数据是你说服管理层扩大投入的最有力武器。

常见 FAQ

"我没时间"

每个练习 ≤5 分钟。用 Copilot 做你本来就要做的事——不是额外任务，是换个方式工作。

"答案不对"

Copilot 不是搜索引擎——给更多上下文、指定来源、追问"再检查一遍"。

"跟我工作无关"

把 [topic] 换成你正在做的事。Journey 教你"怎么让 Copilot 为你工作"，不是让你学某个具体功能。

"结束后呢？"

进入 Champion 社群进阶内容Agent 共创成为下一批 Mentor。
